# Carpeta de Imágenes - NF1 Eventos

Esta carpeta contiene las imágenes y recursos visuales del proyecto.

## Estructura recomendada:

```
img/
├── backgrounds/       # Fondos y texturas
│   ├── hero-bg.jpg
│   ├── field-bg.jpg
│   └── pattern.png
├── icons/            # Iconos personalizados
│   ├── soccer.png
│   ├── trophy.png
│   └── team.png
├── logos/            # Logos y branding
│   ├── logo.png
│   ├── logo-white.png
│   └── favicon.ico
└── players/          # Avatars y fotos de jugadores
    └── default-avatar.png
```

## Tipos de imágenes sugeridas:

### Backgrounds
- **hero-bg.jpg**: Imagen de fondo para el banner principal (1920x1080px)
- **field-bg.jpg**: Cancha de fútbol para secciones (1920x600px)
- **pattern.png**: Patrón de textura sutil para fondos

### Iconos
- Iconos de fútbol, trofeos, equipos
- Formato PNG con transparencia
- Tamaño recomendado: 64x64px o 128x128px

### Logos
- Logo principal de NF1 Eventos
- Versión en blanco para navbar oscuro
- Favicon (32x32px o 16x16px)

## Recursos gratuitos recomendados:

- **Unsplash**: https://unsplash.com/s/photos/soccer
- **Pexels**: https://www.pexels.com/search/football/
- **Freepik**: https://www.freepik.com/free-photos-vectors/soccer
- **Flaticon**: https://www.flaticon.com/ (para iconos)

## Optimización:

- Comprimir imágenes antes de usar (TinyPNG, ImageOptim)
- Usar formatos modernos (WebP) cuando sea posible
- Mantener tamaños de archivo < 500KB para backgrounds
- Usar lazy loading para imágenes grandes
